About

This is natural language processing group project by TomMakesThings, rogerchenrc and laviniafr. This involved building a multi-label classifier trained on a dataset of movie descriptions to predict best matched genres. When given a sample, one to three genres are outputted with a percentage corresponding to its relevance. The model was then saved to file and hosted on a web server.

When developing and evaluating the model, we decided to use the IMDb movies extensive dataset from Kaggle [1]. This dataset has 84983 samples which provides a sufficient number to split into training, validation, and testing. Each film has many attributes, although for this project we will focus on the original title, description and list of one to three genres that can be used as training labels. Across all samples, there are 25 unique genres.
