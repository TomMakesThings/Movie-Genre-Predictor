About

This site explains a natural language processing (NLP) group project done in collaboration with rogerchenrc and laviniafr. In this we tested different NLP pre-processing techniques and model architectures to train a multi-label classifier on dataset of movie descriptions to predict the best fitting genre(s). The state of the top trained model was then saved to file and deployed on a custom built web server as demonstrated in the video below.



Dataset

We decided to use the IMDb movies extensive dataset from Kaggle. This dataset contains 84,983 samples providing a sufficient number to train our classifier. Each film has many attributes, although we focused on the original title, description and list of one to three genres that can be used as training labels. Some pre-processing was therefore required to remove samples missing this information and to remove other attributes.


Classifier

When given a sample, the model outputs one to three genres with a percentage corresponding to its relevance for each one.
